# Tools

1. EternalBlue -  POC which can execute any command on a vulnerable SMBv1 Windows Server 
2. Auto_Recon - a tool to enumerate and automate as much as possible to TMUX windows for multi tasking
3. PowertoBase64 - powershell base64 encoder/ decoder for fast usage
3. mitm_https -  A script to listen on https requests made by a user paralleled to a Deauth for the target.
